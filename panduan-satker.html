Login ke Sistem TC-More

Langkah pertama untuk mengakses aplikasi TC-More adalah melakukan login dengan akun Satuan Kerja yang telah disediakan.

1.1

Akses Halaman Login

Buka browser dan kunjungi halaman login TC-More:

https://dashboard-kementrans.web.app/login.html

Atau akses melalui website utama Kementerian Transmigrasi:

https://www.transmigrasi.go.id/Awasi

Screenshot: Halaman Login TC-More

1.2

Input Kredensial Login

Masukkan kredensial login Satuan Kerja Anda:

Contoh: nakermerauke, nakerposo, nakerjakarta
Format Username Satker
Username menggunakan format: naker[nama_daerah]
Password default: password123
1.3

Verifikasi Login

Setelah berhasil login, sistem akan:

  • Memverifikasi kredensial dengan database Firebase
  • Mengarahkan ke dashboard progress Satuan Kerja
  • Menampilkan data spesifik untuk wilayah kerja Anda
  • Menyimpan session untuk akses selanjutnya
Login Berhasil
Jika login berhasil, Anda akan diarahkan ke dashboard dengan data yang sesuai dengan wilayah kerja Satuan Kerja Anda.

7

Upload Dokumentasi

Panduan lengkap untuk mengupload dokumentasi foto sebelum dan sesudah kegiatan serta dokumen pendukung lainnya.

8.1

Sistem Dokumentasi Link-Based

TC-More menggunakan sistem dokumentasi berbasis link untuk efisiensi storage:

  • Cloud Storage Integration: Google Drive, Dropbox, OneDrive
  • Link-Based Storage: Hemat space server
  • Preview System: Preview langsung dari link
  • Multiple Format Support: JPG, PNG, PDF, DOC, dll
Cloud Storage
Upload file ke cloud storage terlebih dahulu, kemudian input link ke sistem TC-More.
7.2

Form Dokumentasi dalam Edit Kegiatan

Dalam form Edit Kegiatan, terdapat 3 bagian dokumentasi yang dapat diisi:

Upload foto ke Google Drive, Dropbox, dll, lalu paste linknya
Upload foto progress/hasil ke cloud storage
Upload dokumen ke cloud storage, paste link-linknya di sini (satu link per baris)
Preview Otomatis
Sistem akan menampilkan preview gambar dan dokumen secara otomatis setelah link dimasukkan.
7.3

Langkah Upload ke Cloud Storage

Panduan upload file ke cloud storage:

  1. Google Drive:
    • Upload file ke Google Drive
    • Klik kanan file → "Get link"
    • Set permission "Anyone with the link"
    • Copy link dan paste ke form TC-More
  2. Dropbox:
    • Upload file ke Dropbox
    • Klik "Share" → "Create link"
    • Copy link dan paste ke form TC-More
  3. OneDrive:
    • Upload file ke OneDrive
    • Klik "Share" → "Anyone with the link can view"
    • Copy link dan paste ke form TC-More
Permission Setting
Pastikan file permission di-set ke "Public" atau "Anyone with link" agar dapat diakses oleh sistem.
8

Tips & Trik

Best practices dan tips untuk menggunakan aplikasi TC-More dengan maksimal dan efisien.

8.1

Efisiensi Input Data

Tips untuk input data yang efisien:

  • Batch Processing: Kumpulkan data sebelum input untuk efisiensi
  • Template Konsisten: Gunakan format yang konsisten untuk semua input
  • Regular Updates: Update data secara berkala, jangan menumpuk
  • Validation Check: Selalu cek data sebelum submit
  • Backup Data: Simpan copy data penting di local storage
Productivity Tip
Gunakan shortcut keyboard dan auto-fill browser untuk mempercepat input data.
8.2

Mobile Usage

Mengoptimalkan penggunaan di mobile device:

  • Field Updates: Update progress langsung dari lokasi
  • Photo Capture: Ambil foto langsung dengan kamera HP
  • Offline Notes: Catat data offline, sync saat ada koneksi
  • GPS Location: Manfaatkan GPS untuk akurasi lokasi
  • Voice Notes: Gunakan voice recorder untuk catatan cepat
Mobile-First
TC-More didesain mobile-friendly untuk memudahkan akses dari lapangan.
8.3

Troubleshooting Umum

Solusi untuk masalah yang sering terjadi:

Problem: Tidak bisa login
Solution: Cek username format (naker[daerah]), clear browser cache, atau hubungi admin
Problem: Data tidak tersimpan
Solution: Cek koneksi internet, refresh halaman, atau coba browser lain
Problem: Link dokumentasi error
Solution: Pastikan file permission public, cek format link, atau re-upload file
Problem: Dashboard loading lambat
Solution: Clear cache, gunakan browser terbaru, atau akses saat traffic rendah
Technical Support
Jika masalah berlanjut, hubungi technical support atau gunakan fitur help di dalam aplikasi.
8.4

Security Best Practices

Praktik keamanan yang direkomendasikan:

  • Password Security: Ganti password default secara berkala
  • Secure Connection: Selalu akses melalui HTTPS
  • Logout Properly: Selalu logout setelah selesai menggunakan
  • Public Computer: Jangan save password di komputer umum
  • Data Privacy: Jaga kerahasiaan data internal kementerian
Data Security
Data yang diinput adalah data resmi kementerian. Pastikan keamanan dan kerahasiaan data.
